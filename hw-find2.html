請參考本網頁程式碼

使用方式

請輸入要說的話，按下"新增留言"按鈕，即可看到下方出現您剛才輸入的內容。

使用到的jQuery:

.parent( ), .parents( ), .children( ), .val( ), .html( ), .addClass( ), .find( ), .append( )

jQuery程式碼:

$(".reply_Button").click(function(){
var div=$(".reply_Button").parent();
var input=div.children("input");
var value=input.val();
var newReply=$("li").html(value).addClass("reply");
input.val("");
var li=$(".reply_Button").parents(".comment");
var replies=li.find(".replies")
replies.append(newReply);
});
