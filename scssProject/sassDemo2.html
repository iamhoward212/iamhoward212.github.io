SCSS實例 :


  • 在scss檔案夾中 有四個.scss檔案

    1. tool.scss 的內容
    請參考程式碼

    @import 'variable.scss';
    @import 'mixin.scss';
    @import 'content.scss';

    2. _content.scss 的內容
    請參考程式碼

    .square{
    color:$square-color;
    border:1px solid black;
    @include height(300);
    @include width(200);
    @include bg-color;
    @include add-border-radius(10);
    }
    .circle{
    color:$circle-color;
    }

    3. _mixin.scss 的內容
    請參考程式碼

    @mixin add-border-radius($radius){
    border-radius:#{$radius}px;
    }
    @mixin bg-color{
    background-color:brown;
    }
    @mixin height($height){
    height:#{$height}px;
    }
    @mixin width($width){
    width:#{$width}px;
    }

    4. _variable.scss 的內容
    請參考程式碼

    $square-color:red;
    $circle-color:green;

    scss轉譯成css後 可以得到CSS資料夾裡的tool.css

    CSS資料夾的 tool.css 內容如下
    請參考程式碼

    .square {
    color: red;
    border: 1px solid black;
    height: 300px;
    width: 200px;
    background-color: brown;
    border-radius: 10px;
    }
    .circle {
    color: green;
    }
